About {{ page.title }}

This is a beta version of a bead crochet design tool. It applies ideas from the design framework described in the book {{ page.book }} by Ellie Baker and Susan Goldstine. It was created as a supplement to an upcoming new edition of the book. The software was developed by Ellie Baker and Scott Vorthmann.

Use it at your own risk, since it is still in development, but stay tuned for more info and instructions soon. Feedback may be sent to craftingconundrums@gmail.com.

How to use {{ page.title }}

Click on any "bead" (circle) in the Vertical Repeat or Bead Plane to set its color, using the currently highlighted swatch under Bead Color.

All actions affecting the Repeat Length, Circumference, and the bead colors in the Vertical Repeat can be undone (and redone), so feel free to play!
